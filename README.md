# StreamHub

StreamHub is a web-based IPTV streaming website that allows playback of M3U and M3U8 streams directly in the browser.

This repository is currently used for testing and development purposes.

## Features
- Play M3U8 streams
- Load M3U playlists
- Browser video player
- Modular JavaScript structure

## Tech Stack
- HTML
- CSS
- JavaScript
- HLS.js

## How to Run
Open `index.html` in your web browser.
