\# StreamHub



StreamHub is a lightweight web-based IPTV streaming website that plays M3U and M3U8 streams directly in the browser.



This project is currently under development and used for experimentation and feature trials.



\## Features (Work in Progress)



\* M3U8 stream playback

\* M3U playlist parsing

\* Browser-based video player

\* Modular JavaScript structure



\## Tech Stack



\* HTML5

\* CSS3

\* Vanilla JavaScript

\* HLS.js



\## Run Locally



Open `index.html` in a web browser.
